## 技能简介

CRRC-PPT 是一个专门用于生成中国中车（CRRC）企业视觉风格的 PowerPoint 演示文稿的 AI 技能。该技能采用 HTML 到 PPTX 的转换方案，通过 `dom-to-pptx` 库实现高质量的幻灯片导出。

## 核心特性

### 1. 视觉风格
- **品牌主色**: 中车红 (#C70019)
- **设计风格**: 央企工业咨询风格，稳重、清晰、克制
- **布局系统**: Bento Grid 响应式网格布局
- **比例**: 16:9 标准演示比例

#### 配色方案

本技能提供两种卡片配色方案，可在 Step 1 确认时选择：

**方案 A：中车红灰配色（默认）**
- 主色调：白底 + 灰边框
- 强调色：中车红 (#C70019) 用于关键信息
- 风格：稳重、专业、克制
- 适用场景：正式汇报、战略规划、高层展示

**方案 B：彩色配色**
- 多色卡片：使用色板中的多种颜色
- 视觉活力：更丰富的色彩层次
- 风格：现代、活泼、多元
- 适用场景：创新项目、团队分享、多主题内容

> **默认行为**：如用户在 Step 1 未明确要求，系统默认采用**方案 A**。

### 2. 工作流程
采用**两步确认流程**，确保内容准确性：

**Step 1: 卡片拆解与布局确认**
- 内容结构化分析
- 模块层级规划（主卡/辅助卡/数据卡）
- 布局方案设计
- 配色方案选择（A 或 B）
- 用户确认闭环

**Step 2: HTML 生成**
- 严格遵循设计规范
- 应用用户确认的配色方案
- PPTX 兼容性保证
- 自动合规检查
- 一键导出功能

### 3. 质量保证
- 内容保真优先
- 标题禁词检查
- 空间评估机制
- 自动化验证脚本

## 使用方法

### 基本用法

在 Claude Code 中调用技能：

```
/crrc-ppt [你的内容或需求]
```

### 工作流示例

1. **提供内容**：将需要制作成 PPT 的文字内容提供给技能
2. **确认结构**：AI 会先输出卡片拆解方案，包含配色方案说明
3. **选择配色**（可选）：
   - 默认使用方案 A（中车红灰配色）
   - 如需方案 B（彩色配色），在确认时说明
4. **生成 HTML**：确认后生成完整的 HTML 幻灯片
5. **导出 PPTX**：在浏览器中打开 HTML，点击"Export PPTX"按钮

### 配色方案选择示例

```
# 使用默认方案 A
用户："确认，按这个方案生成"

# 切换到方案 B
用户："确认，但使用方案 B 彩色配色"
```

## 设计原则

1. **内容保真 > 可读性 > 逻辑结构 > PPTX 可转换性 > 品牌视觉 > 装饰效果**
2. 标题必须来自原文关键词，避免空泛概念词
3. 中车红仅用于关键强调，使用需克制
4. 每页必须有明确的视觉锚点
5. 信息装不下时采用拆页/减卡/合并等手段，严禁降低字号

## 技术栈

- **前端**: HTML5 + CSS3
- **布局**: Flexbox + CSS Grid (Bento Grid)
- **导出**: dom-to-pptx (最新版本)
- **字体**: 微软雅黑 (Microsoft YaHei)
- **兼容性**: 针对 PowerPoint 2016+ 优化

## 适用场景

### 方案 A 适用场景
- 企业正式汇报
- 战略规划文档
- 高层管理展示
- 对外商务演示

### 方案 B 适用场景
- 创新项目展示
- 团队内部分享
- 多主题内容整合
- 年轻化受众演示

## 注意事项

1. 必须完成 Step 1 确认后才能进入 Step 2
2. 不得跳过用户确认环节
3. 配色方案一旦在 Step 1 确认，Step 2 必须严格执行
4. 严格遵循参考文档中的规范约束
5. 优先保证内容完整性和可读性
6. 导出前务必进行自检验证
